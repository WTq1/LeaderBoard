Problem 4 - Solving a System of Equations

To find the values of x and y, we can solve the given system of equations:

Equation 1: 2x + 3y = 7

Equation 2: 4x - y = 1

We can solve this system using the method of substitution or elimination. Let's use the method of substitution.

From Equation 2, we can express y in terms of x:

y = 4x - 1

Now, substitute this value of y into Equation 1:

2x + 3(4x - 1) = 7

Expand and simplify the equation:

2x + 12x - 3 = 7

Combine like terms:

14x - 3 = 7

Add 3 to both sides:

14x = 10

Divide both sides by 14:

x = 10/14

Simplify the fraction:

x = 5/7

Now, substitute this value of x into Equation 2 to find y:

4(5/7) - y = 1

20/7 - y = 1

Subtract 20/7 from both sides:

-y = 1 - 20/7

Combine the fractions on the right side:

-y = (7 - 20)/7

-y = -13/7

Multiply both sides by -1:

y = 13/7
