Problem 5 - Inverse Variation

To find the cost of printing 250 books, we can use the concept of inverse variation.

According to the problem, the cost of printing a certain number of books varies inversely with the number of books printed.

We can set up the inverse variation equation as:

C = k/n

Where C represents the cost, k is the constant of variation, and n is the number of books printed.

We are given that it costs $200 to print 100 books. We can use this information to find the value of k.

Substituting the values into the equation:

200 = k/100

Multiply both sides by 100:

200 * 100 = k

k = 20000

Now that we have the value of k, we can find the cost of printing 250 books.

Substituting the values into the equation:

C = 20000/250

C = 80

Answer: It will cost $80 to print 250 books.
